## Данные и конфиденциальность

### Собираемая информация
1. **Данные Telegram**:
    - ID чатов
    - Запросы и временные метки
    - Статистика использования команд

2. **Данные о товарах**:
    - Кэшированная информация о товарах Wildberries (цены, рейтинги)
    - История поисковых запросов
    - Результаты поиска из Google Shopping

### Хранение
- Данные хранятся в MongoDB (размещённой на MongoDB Atlas).
- Кэшированные данные о товарах хранятся **2 часа** (настраивается).
- Метрики команд хранятся неограниченное время для аналитики.

### Сторонние сервисы
- **API Wildberries**: Запросы регулируются [Условиями использования API Wildberries](https://www.wildberries.ru/services/api).
- **Google Shopping**: Используется через парсинг веб-страниц в соответствии с [Политикой использования Google](https://policies.google.com/terms).
