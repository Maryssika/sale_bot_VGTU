# Имя проекта
Sale_bot_VSTU

# Описание
Sale_bot_VSTU — это Telegram-бот для мониторинга цен на товары с маркетплейсов Wildberries и Google Shopping. Он предоставляет:

- Поиск товаров по запросу.

- Просмотр истории цен за разные периоды (день, неделя, месяц).

- Прогнозирование цен на 7 дней.

- Кэширование данных для ускорения повторных запросов.

# Проблемы, которые решает проект:

- Отсутствие удобного инструмента для отслеживания динамики цен.

- Сложность сравнения цен на разных площадках.

- Необходимость ручного сбора данных для анализа.

# Технологический стек
Язык программирования: Java

# Фреймворки и библиотеки:

- Telegram Bot API (telegrambots)

- MongoDB (хранение данных)

- Jsoup (парсинг Google Shopping)

- HTTP-клиент (для работы с Wildberries API)

- Клонировать репозиторий:
<https://github.com/Maryssika/sale_bot_VGTU.git>.

- Установить MongoDB и запустить сервер.

- Настроить подключение к базе данных в MongoDBService.

# Применение
- Запустите бота в Telegram.

- Используйте команды:

/wb [запрос] — поиск на Wildberries.

/google [запрос] — поиск в Google Shopping.

/cacheinfo [запрос] — информация о кэше.

/forecast [запрос] — прогноз цен на 7 дней.

# Проверка ПО
Автоматизированные тесты пока отсутствуют.

# Проблемы
- Ограниченная поддержка маркетплейсов (только Wildberries и Google Shopping).

- Прогнозирование цен основано на упрощенной логике.
